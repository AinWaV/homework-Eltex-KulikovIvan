Данная программа (калькулятор) подгружает плагины из папки plugins, имя плагина должно быть типа: plugin_* . 

Заголовочный фаил pluginFunctions.h описывает общие функции плагинов, с помощью которых происходит взаимодествие с ними из под основной программы main.bin

Фаил pluginsUpload.c содержит код функции, которая выполняет загрузку плагинов из указанной директории, а также выполняет проверку плагина на совместимость с программой, в данную функцию передается массив указателей на хенделы плагинов, который будет заполняться по мере подгрузки плагинов, а также передается для заполнения переменная-счетчик кол-ва подгруженных плагинов.

FilesInFolder.c содержит функцию которая должна определить имена и количество файлов в указанной директории. Функция заполняет переданную в нее структуру namesfilesInDir, объявленную в файле FilesInFolder.h, содержащую в себе поля: путь к директории поиска файлов, переменную счетчик кол-ва файлов в директории, а также массив указателей на строки (имена файлов)

main.с запускает функции подгрузки плагинов, а также функцию mainMenu, которая печатает менюшку в консоль, ожидает ввод действия от пользователя, опрашивает плагины и выдает о них информацию, запускает функцию расчета из требуемого плагина.
