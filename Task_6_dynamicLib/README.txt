Задача: Подгрузить динамическую библиотеку и воспользоваться функциями из нее :)

1. Создаем объектные файлы (с ключем -fPIC (относительная адресация)):
gcc -fPIC -c func*.c

2.Для того, чтобы создать динамическую библиотеку надо использовать ключ -shared:
gcc -shared -o libArithmeticDyn.so func*.o
rm *.o

3. Компиляция программы
gcc main.c -o calc.bin -L. -lArithmeticDyn -ldl
ps Чтобы получить список динам библиотек, которые использует приложение исп утилиту ldd [имя приложения]

4. Перед запуском также не забываем добавить нашу директорию с библиотекой в список известных директорий надо подредактировать файл /etc/ld.so.conf
Лучше всего сделать так:
sudo ldconfig ~/Desktop/Task№6_dynamicLib

5. Запускаем бинарник
